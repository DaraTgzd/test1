# My New Repo

This repository contains a simple Python script `draw_circles.py` that draws a number of circles on the screen using turtle graphics.

## Usage

Run the script with Python and provide the number of circles as a command-line argument or interactively when prompted.

```bash
python3 draw_circles.py 5  # draws five circles
```

If no argument is provided, the script will ask for the number of circles.
